# 🖥️ DGM Tech

Sistema de cadastro e gerenciamento de estoque voltado para peças de computador.

## 📌 Sobre o Projeto

O **DGM Tech** é um sistema desenvolvido com o objetivo de facilitar o controle de estoque de componentes internos de computadores, permitindo o cadastro, organização e busca eficiente dos produtos.

Diferente de sistemas genéricos, o projeto é focado exclusivamente em **peças de hardware (não periféricos)**, como:
- Processadores
- Placas-mãe
- Memórias RAM
- Placas de vídeo
- Armazenamento (HDs/SSDs)
- Fontes

## 🚀 Funcionalidades

- 📥 Cadastro de produtos
- 📋 Listagem de itens no estoque
- 🔍 Busca por categoria
- 🗂️ Organização dos produtos por tipo
- ✏️ (Opcional) Edição de produtos
- ❌ (Opcional) Remoção de produtos

## 🛠️ Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- PHP

## 🎯 Objetivo

O projeto foi desenvolvido com fins educacionais, visando praticar:
- Lógica de programação
- Manipulação de DOM
- Organização de dados
- Trabalho em equipe
